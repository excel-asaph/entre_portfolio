## Getting Started

To run this project locally:

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  Open your browser at the URL shown in the terminal (usually `http://localhost:5173`).
